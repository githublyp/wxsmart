/*微信小程序开发
本文档搭建微信小程序的基本目录，页面框架，常用组件
目录结构
image：图片资源，大小不要太大，最好用在线的图片；
pages：页面内部包含
    .js 页面逻辑层
    .wxml 页面视图模板
    .json 页面配置对象，主要是窗口配置
    .wxss 页面样式
utils：工具函数库
app.js：程序入口文件，初始化程序运行所需页面，及全局数据，方法
app.json：程序配置文件，
    pages：页面列表
    window：窗口配置
    tabBar：底部标签
    networkTimeout：网络配置
    debug：调试模式
app.wxss：程序全局样式表，可通过@import引入
*/
